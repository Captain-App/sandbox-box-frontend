The Workspace is where you interact directly with your agent.

![Workspace](/screenshots/workspace.png)

### Features

#### 1. Chat Panel

A conversation-based interface for giving instructions to your agent. You can see real-time progress and logs.

#### 2. Preview Frame

A live, hosted preview of what your agent is building. If the agent deploys a web app or API, it will appear here.

#### 3. Control Bar

- **Back to Dashboard**: Return to the main overview.
- **Open in New Tab**: View the hosted preview in a full browser tab.
- **Fullscreen**: Toggle the preview frame to take up the whole screen.
