The Boxes view lists all your current and past sandboxes.

![Boxes List](/screenshots/boxes.png)

### Managing Boxes

From this view, you can:

- **Select** a box to make it active on the dashboard.
- **Open** a box to enter the [Workspace](/features/workspace).
- See usage statistics for the current month.

### Creating a New Box

Click **Create New Box** to initialize a new environment.

![Create Sandbox](/screenshots/create-sandbox.png)

Specify:

- **Title**: A friendly name for your box.
- **Region**: Choose the deployment region (e.g., LHR, SFO).
- **Repository**: (Optional) Provide a GitHub repo to clone on startup.
