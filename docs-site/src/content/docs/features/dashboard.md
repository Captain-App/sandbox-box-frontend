The Shipbox Dashboard is your command center for managing agent environments.

![Dashboard](/screenshots/dashboard.png)

### Key Components

#### 1. Sandbox Status

Monitor your active sandboxes, their region, and current status (active, idle, or stopped).

#### 2. Credit Balance

Shipbox uses a credit-based system. You can see your available credits at a glance.

#### 3. Activity Timeline

A real-time feed of what your agents are doing across all boxes.

#### 4. Navigation Sidebar

Quickly switch between:

- **Dashboard**: Main overview.
- **Boxes**: Detailed list of all sandboxes.
- **Activity**: Full history of agent logs.
- **Billing**: Manage credits and payments.
- **Settings**: Configure API keys and integrations.
