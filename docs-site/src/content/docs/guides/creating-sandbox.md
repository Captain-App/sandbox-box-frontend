title: Creating Sandboxes
description: Step-by-step guide to creating new agent environments.
---

Creating a new sandbox is the first step to giving your agent a workspace.

## Using the Dashboard

1. Navigate to the **Boxes** view or click **Create New Box** on the dashboard.
2. Fill in the sandbox details:
   - **Title**: Choose a descriptive name.
   - **Region**: Select the region closest to your users or repository.
   - **Repository**: (Optional) Enter a GitHub URL to clone code into the sandbox automatically.

## Regional Availability

Shipbox currently supports the following regions:

- `lhr`: London, UK
- `sfo`: San Francisco, US
- `hkg`: Hong Kong, HK
